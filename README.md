# TikTok Automation Bot

## Quick Start Guide

### 1. Install Python
Download from python.org (3.9 or higher)

### 2. Install Dependencies
**Windows:** Run `setup.bat`
**Mac/Linux:** Run `bash setup.sh`

### 3. Configure
Edit `config.yaml`:
- Add your email and password
- Set target username
- Adjust delays if needed

### 4. Add Comments
Edit `input/comments.txt` - one comment per line

### 5. Run Bot
`python main.py`

## Safety Tips
- Use a secondary account
- Start with small delays (60+ seconds)
- Monitor your account daily
